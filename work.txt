Full cinema player brain. Most complex controller.

State:
_player          → media_kit Player instance (64MB buffer)
_currentEp       → current episode index
_isLandscape     → whether screen is rotated
_showControls    → whether controls are visible
_isLocked        → screen locked (no touch except long press)
_showDrawer      → episode drawer open/closed
_showToolsDrawer → tools drawer open/closed
_speed           → playback speed (0.5x to 2x)
_position        → current playback position
_duration        → total duration of current episode
_isBuffering     → buffering indicator
_videoAspectRatio → detected from actual video stream
_isVerticalVideo  → true if aspect ratio < 1.0

init():
→ enables wakelock (screen stays on)
→ opens first episode URL
→ listens to videoParams stream (detects aspect ratio)
→ listens to position stream (updates seek bar)
→ listens to duration stream (updates time label)
→ listens to buffering stream (shows spinner)
→ listens to completed stream (auto next episode)

playEpisode(index):
→ opens new R2 URL for that episode
→ resets position/duration
→ starts playing immediately

Key methods:
togglePlayPause()    → play or pause
seekRelative(secs)   → jump ±10 seconds
seekProgress(0.0-1.0) → seek bar position
setSpeed(double)     → change playback speed
toggleOrientation()  → rotate portrait/landscape
toggleLock()         → lock/unlock screen
skipIntro()          → jump to 90 seconds
toggleDrawer()       → open/close episode list
toggleToolsDrawer()  → open/close tools menu
showControlsTemporary() → show controls then auto hide after 4s

dispose():
→ disables wakelock
→ resets orientation to portrait
→ disposes player

Auto builds APK on every push to main/master branch.

Steps:
1. Checkout code from GitHub
2. Setup Java 17 (required by Flutter for Android)
3. Setup Flutter 3.22.0 stable
4. flutter pub get (install dependencies)
5. flutter build apk --release --split-per-abi
6. Upload APKs as downloadable artifact (7 days)

--split-per-abi generates 3 APKs:
→ arm64-v8a   ← use this (all modern Android phones)
→ armeabi-v7a ← older phones
→ x86_64      ← emulators only

Download from: GitHub → Actions tab → latest run → Artifacts

CHANGE HERE IF:
→ Upgrading Flutter version
→ Changing artifact retention days
→ Adding iOS build (needs macOS runner)
→ Adding automated testing step

HOW ALL FILES CONNECT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

main.dart
└── provides SettingsController globally
└── launches FeedScreen

FeedScreen
└── creates + provides FeedController
└── IndexedStack → Feed | Explore | Profile

FeedController
└── calls ApiService.getFeed()
└── creates Player per trailer URL
└── provides Player to FeedCard via getPlayer()

FeedCard
└── reads Player from FeedController → VideoController
└── shows thumbnail + trailer
└── on tap Watch → ApiService.getMovie(slug)
└── creates PlayerController with real episodes
└── pushes PlayerScreen

PlayerController
└── opens episode URLs from R2
└── detects aspect ratio from stream
└── auto next via stream.completed
└── reads SettingsController.autoNext

PlayerScreen
└── reads PlayerController
└── reads SettingsController
└── _VideoArea decides vertical/horizontal layout
└── _EpisodeDrawer shows real episodes from controller
└── _ToolsDrawer reads/writes both controllers

ExploreScreen
└── calls ApiService.search()
└── reads FeedController.movies for trending
└── on tap → ApiService.getMovie() → PlayerScreen

ProfileScreen
└── reads/writes SettingsController
└── shows static profile (connect real auth later)
